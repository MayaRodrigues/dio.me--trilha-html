Surgimento da internet

A internet é uma ampla rede que conect milhoes de pessoas, computadores e dispositivos(celulares, tablets...) por todo o mundo. Ela nos permite acessar informações, comunicar-se com outras pessoas e realizar diversas atividades online. A internet é baseada em um conjunto de protocolos que permitem a troca de dados entre dispositivos conectados.

Como funciona a Internet?

A Internet funciona conectando redes através de uma série de roteadores e switches. Um roteador encaminha pacotes de dados entre diferentes redes, enquanto um switch vincula dispositivos em uma única rede. Isso permite que os computadores se comuniquem e acesse o conteúdo armazenado em servidores remotos.

Como funciona o HTML

HTML (HyperText Markup Language) é a linguagem base da web — usada para criar páginas e organizar o conteúdo que vemos nos navegadores. Não é linguagem de programação, e sim de marcação , ou seja, ela define a estrutura e o significado do conteúdo.

Com HTML você não cria lógica de sistemas, mas sim organiza e dá significado à informação que será apresentada ao usuário.

O que são tags

No HTML, tags são os comandos que usamos para dizer ao navegador como apresentar determinado conteúdo. Cada tag tem uma função específica, como criar títulos, parágrafos, listas ou links.

No primeiro módulo, aprendi várias tags fundamentais que vão me acompanhar em toda a jornada como desenvolvedora.

Exemplos de tags aprendidas

  • <p> — Cria um parágrafo.
  • <h1> até <h6> — Criam títulos com diferentes níveis de importância.
  • <a> — Cria um link para outra página ou site.
  • <ul> e <ol> — Criam listas (não ordenadas e ordenadas).
  • <li> — Define os itens dentro das listas.
  • <mark>, <strong>, <small>, <i>, <u> — Alteram a aparência ou destaque do texto.

Praticando com estrutura semântica

Além das tags básicas, comecei a entender a importância de HTML semântico, que organiza melhor a informação e melhora a acessibilidade. Já usei <main>, <header>, <nav> e <section> em pequenos testes e fiquei fascinada com a ideia de que até os robôs entendem melhor nosso site quando usamos HTML corretamente.
